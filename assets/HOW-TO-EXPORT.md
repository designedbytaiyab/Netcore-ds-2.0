# How to Export Screen Images

Drop your Figma exports into this folder using the exact filenames below. Once the images are here and pushed to GitHub, Claude Code will load and visually process them when the matching trigger fires.

**Export settings in Figma:** Select the frame → Export → PNG → 1x (or 2x for retina) → Export

## Brand Assets
Place these from your `Logo___Brand_Assets.zip`:

| Filename | What it is |
|---|---|
| `netcore-logo.svg` | Primary Netcore logo (horizontal) |
| `netcore-logo-white.svg` | White version for dark backgrounds |
| `netcore-icon.svg` | Icon-only mark (no wordmark) |
| `netcore-logo-dark.svg` | Dark version |
